## 🚀 Project Overview

This project simulates **real-world surveillance conditions** where images are degraded due to noise (low light, weather, transmission errors).

We:
- Add artificial noise to images  
- Apply restoration filters  
- Evaluate performance using metrics  

---

## 🎯 Objectives

- Understand image noise types  
- Implement Gaussian and Salt & Pepper noise  
- Apply spatial filters for restoration  
- Evaluate quality using MSE and PSNR  
- Compare filter performance
