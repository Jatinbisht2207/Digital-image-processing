##  Project Overview

Medical imaging systems (X-ray, MRI, CT) generate large volumes of data.  
This project focuses on:

-  Compressing medical images using Run Length Encoding (RLE)  
-  Segmenting important regions (organs, bones, etc.)  
- Refining segmentation using morphological operations  

---

##  Objectives

- Understand image redundancy and compression  
- Implement Run Length Encoding (RLE)  
- Perform image segmentation using thresholding  
- Improve segmentation using morphological operations  
- Analyze results for medical relevance
